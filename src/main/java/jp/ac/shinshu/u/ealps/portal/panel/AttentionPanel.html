eALPSを利用する上での注意事項
  •  eALPSをご利用のみなさま
    • 信州大学で推奨しているブラウザはGoogle ChromeとFirefoxです．eALPSを利用する際も信州大学が推奨するこれらのブラウザをご利用ください．
    • ※ 時間割・eALPS内のコンテンツ等が正しく表示されない場合には，推奨ブラウザでお試しください．
    • ※ Internet ExplorerおよびMicrosoft Edgeでは正常に表示されない場合がありますのでご注意ください．
  •  教職員のみなさま
    • 上記コース一覧にご自身の担当するコースが表示されない場合は，e-Learningセンターまでご連絡ください．また，不要なコースが記載されている場合にもご相談ください．
    • 同一内容のコースを複数運用する場合には，便利な運用方法があります．コースコンテンツ作成前にe-Learningセンターまでご連絡ください．
    • 旧カリキュラムを履修登録する学生で，新カリキュラムコースに読み替えて登録が必要な場合にはe-Learningセンターまでご連絡ください．
    • 教員，TA，CP をコースに追加登録する場合はe-Learningセンターまでご連絡ください．
    • 学生は履修登録により自動的にコースに登録されます．履修登録に関係なく，追加・削除が必要な場合にはe-Learningセンターまでご連絡ください．
  •  教職員及び学生のみなさま
    • 日本語（全角）を含む名前のつけられたファイルをeALPS上にアップロードすると，そのファイルを閲覧できない場合があります．eALPSにアップロードするファイル名は，半角英数字を使用してください．
    • 現時点で，Android端末でPDFファイルが正しく表示できない場合があります．PDFファイルを一旦端末にダウンロード（ファイル名を長押し→「新しいタブで開く」）していただくと閲覧できる場合がありますので，お試しください．
  •  教材等の無断転用は禁止されています
    • 教材やデータは著作物であり，教員等その著作権者の許可を得ずに転用することは禁止されています．しかし，eALPS等で提供している教材コンテンツ等を無断で外部サーバ等に転用しているケースが見られました．そのような行為が見られた場合には，著作権者は転用者に対して対処をする場合もあります．
